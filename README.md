# Stopwatch Web Application

This repository contains the solution for **Task 02** of the SkillCraft Technology internship, as shown in the task assignment document IMG_8611.jpg. The project is an interactive, clean, and user-friendly stopwatch web application built to accurately measure and record time intervals.

## 🚀 Project Features

* **Interactive Controls:** Seamless functionality to start, pause, and reset the stopwatch tracking.
* **Lap Timing System:** Dynamically tracks, records, and displays individual lap times in a structured list.
* **User-Friendly Interface:** A clean, responsive dashboard designed for optimal visibility and ease of use across different device screen sizes.

## 🛠️ Technologies Used

* **HTML5:** Structures the semantic layout of the application components and controls.
* **CSS3:** Handles modern UI styling, responsive design layouts, and intuitive button hover/active states.
* **JavaScript (ES6):** Manages the precise interval timing logic, button states, and dynamic DOM manipulation for lap creation.
